# CampusHub – Student Service Platform

CampusHub is a web-based platform that allows students to:
- Find tutors
- Offer tutoring services
- Connect with peers for academic help

## Features
- Tutor listing system
- Form validation (name, subject, contact)
- Clean UI using HTML/CSS
- Backend powered by Flask
- SQLite database integration

## Tech Stack
- Python (Flask)
- HTML, CSS
- SQLite

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:500
